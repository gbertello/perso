## Perso

This repository has been created to store source code of my personal site.

## Prerequisites

Install docker. Use this command on Ubuntu 18.04:

```
snap install docker
```

## Installing

Follow these instruction to deploy the docker container and serve the website.

```
chmod +x run.sh
./run.sh
curl http://localhost
```
